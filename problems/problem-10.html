Project Euler with Javascript

Problem 10 description

The sum of the primes below 10 is 2 + 3 + 5 + 7 = 17.

Find the sum of all the primes below two million.

My solution

Note: This solution uses the "isPrime()" function as defined in problem 7 

			
	function problem10(number) {
		var answer = 0;
		
		for( i = 2; i < number; i++ ) {
			if ( isPrime(i) ) {
				answer += i;
			}
		}
		
		return answer;
	}
	
	problem10(2000000);
